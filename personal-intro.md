## 关于我

我是 DongZi（张东炜），一名初中生独立开发者，同时也是 VEX IQ 赛队队长。我长期围绕 AI 工具链、Web 开发、交互设计、平面动画与机器人教育开展个人项目。

相比"刷题式学习"，我更偏向项目驱动学习。我习惯通过真实项目快速掌握新工具，并尝试把设计、技术与表达整合到同一个作品里。

### 核心理念

> **做"真正能被使用的作品"，而不只是"完成任务的项目"。**

很多人做项目像交作业——做完、截图、结束。我更在意产品是否真的完整、视觉是否统一、用户是否愿意继续使用、项目是否具有长期迭代价值。技术只是工具，表达、体验与思考同样重要。

## 项目作品

### [Numbers-In-Pi](https://github.com/flashzdw/Numbers-In-Pi)

在圆周率中查找任意数字，并搭配精美纪念图片的趣味网站。

**技术特点**

- React + Vite 前端，Cloudflare Workers 后端
- 自研索引算法实现快速数字检索
- GitHub Releases 管理静态数据文件
- 浏览器自动化测试保障部署质量

**项目亮点**：用算法解决趣味问题，将数学探索与视觉表达结合。

---

### [HTML-Shower](https://html.novecho.qzz.io) · [源码](https://github.com/flashzdw/HTML-Shower)

手机端纯前端 HTML 实时预览工具。

**技术特点**

- 支持 PWA——可添加到主屏幕、离线使用
- 纯前端实现，不上传用户代码，保护隐私
- Playwright 自动化测试覆盖
- GitHub Actions 持续部署

**项目亮点**：解决移动端调试 HTML 的痛点，专注工具化体验。

---

### [BricksWorkshop-Web](https://www.vexiq8009.website) · [源码](https://github.com/flashzdw/BricksWorkshop-Web)

VEX 教育机构展示型官网

**项目特点**

- 一页式平滑滚动设计，极简全白背景
- 彩虹渐变视觉点缀，强调现代教育科技感
- 高度统一的设计语言，移动端与桌面端响应式适配

**独立完成**：网站策划、UI 设计、前端开发、动效设计、内容组织、AI 协同开发流程

**项目价值**：打破传统教育机构网站"科技蓝 + 信息堆砌"的设计模式，以年轻化、现代化与产品感为核心表达。
